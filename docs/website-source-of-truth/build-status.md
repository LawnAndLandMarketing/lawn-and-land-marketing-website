## Success Criteria For “Finished” Pages
A page should not be treated as finished just because it is live.
A finished page should have:
- a clear strategic purpose
- a strong H1 and promise
- enough body content to persuade and rank
- proof / trust signals where appropriate
- clear CTA paths
- correct canonical internal linking
